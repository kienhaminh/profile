## Overview

The Admin Analytics API provides comprehensive statistics and insights for the admin dashboard, including content metrics, user engagement data, and system analytics.

## Dashboard Statistics

### Get Dashboard Stats

Retrieve comprehensive statistics for the admin dashboard.

```http
GET /api/admin/stats
```

**Response:**

## Metrics Breakdown

### Posts Over Time
- **Timeframe**: Last 12 months
- **Grouping**: By month and status
- **Purpose**: Track content creation velocity and publishing patterns

### Topics Distribution
- **Scope**: Top 10 topics by post count
- **Purpose**: Understand content categorization patterns

### Hashtags Distribution
- **Scope**: Top 10 hashtags by total usage (posts + projects)
- **Purpose**: Identify trending topics and popular tags

### Projects Statistics
- **Grouping**: By publication status
- **Purpose**: Monitor project completion and publication status

### Recent Activity Summary
- **Timeframe**: Current week + all-time totals
- **Metrics**: Recent creation counts and total content counts
- **Purpose**: Quick overview of recent activity and system scale

## Usage Examples

### Track Content Growth
```javascript
const stats = await fetch('/api/admin/stats');
const data = await stats.json();

// Calculate monthly growth
const monthlyPosts = data.postsOverTime
  .filter(item => item.status === 'PUBLISHED')
  .map(item => item.count);

// Display recent activity
console.log(`${data.recentActivity.posts_this_week} posts created this week`);
```

### Identify Popular Topics
```javascript
const topTopics = data.topicsDistribution
  .sort((a, b) => b.post_count - a.post_count)
  .slice(0, 5);

console.log('Top 5 topics:', topTopics.map(t => t.name));
```

## Authentication Required

This endpoint requires admin authentication via HTTP-only cookie.

## Error Responses

**401 Unauthorized:**
```json
{
  "error": "Unauthorized"
}
```

**500 Internal Server Error:**
```json
{
  "error": "Failed to fetch dashboard statistics"
}
```

## Performance Notes

- Statistics are computed on-demand from the database
- Consider caching for high-traffic dashboards
- Large datasets may impact response time
- All queries are optimized with appropriate indexes
